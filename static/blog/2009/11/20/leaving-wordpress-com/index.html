Leaving Wordpress.com

20 November 2009

I don't like wordpress taking ownership of links http://en.forums.wordpress.com/topic/link-redirection-through-go2wordpresscom?replies=12#post-404181 so I'm breaking off. I use wordpress partly as a link pastebin and archive so not being able to just copy-n-paste links is a real deal breaker for me. I wonder which of these links will be replaced into go2 jacked links: http://www.youtube.com http://www.cnn.com http://www.blizzard.com http://www.thebestpageintheuniverse.net/c.cgi?u=swine_flu Hurrah for the export/import tool.
